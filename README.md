# Micro Rails Blog

A micro blog implemented with Ruby on rails.

## Getting Started

Clone the repository. Then run these commends from your terminal.
1. rake db:migrate
2. bundle install
3. rails s
4. browse to localhost:3000 and signup.



## Built With

* [Ruby on Rails](http://rubyonrails.org/)
